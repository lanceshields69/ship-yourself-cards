# Ship Yourself Cards 

Ship Yourself Cards is a helpful card deck app researched, designed, and coded using AI tools to help designers of all levels build momentum, tell their story, and navigate career transitions with confidence.

Built with: V0.dev, Figma, Chatgpt

https://www.shipyourselfcards.com

---

## Features

- 7 Categories of actionable cards:
  - Mindset & Resilience
  - Networking Moves
  - Super IC Designer
  - Interview Gameplans
  - Storytelling + Personal Brand
  - Offer/Negotiation Tactics
  - Wild Cards (Power-Ups, Pep Talks)
- Shuffle + Ship: Randomize card draws to spark new ideas
- Responsive design: Mobile and desktop friendly
- Clean, focused UI for inspiration and tactical momentum
